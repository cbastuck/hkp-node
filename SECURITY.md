# Security

## Reporting a Vulnerability

If you discover a security vulnerability in hkp-node, please report it by email to **[mail@cbastuck.de](mailto:mail@cbastuck.de)**.

**Please do not open a public issue for security vulnerabilities.** Public disclosure before a fix is available puts all users at risk.

Include in your report:

- A description of the vulnerability and what an attacker could achieve
- Steps to reproduce (proof-of-concept code or a minimal example)
- The version(s) affected

## Response

I will acknowledge your report within **5 business days** and aim to provide a fix or mitigation within **30 days**, depending on severity and complexity. I will credit you in the release notes unless you prefer to remain anonymous.

## Scope

This repository covers the hkp-node runtime server. For vulnerabilities in the Hookup web application or other components, use the same email address.

## Out of Scope

- Vulnerabilities in third-party dependencies (report those upstream)
- Issues in applications built *with* hkp-node by third parties — contact those operators directly
